## 2. 供电电路

> ⚠️ 电机标称 12V，建议使用 **3 节 18650 串联**（满电 ~12.6V，标称 ~11.1V）保证电机转速和扭矩。

```
18650 电池组（3 节串联，~11.1V）
        │
        ├──[ON/OFF 拨动开关]──→ 2M16S  VIN / VM（电机电源输入）
        │                              │
        │                     2M16S 板载 5V 稳压（LDO 或开关稳压）
        │                       ├── 5V ──→ YFRobot 2015 VCC
        │                       ├── 5V ──→ Digital PWM UNO 5V
        │                       ├── 5V ──→ 激光发射器 VCC
        │                       └── 5V ──→ VS1838B VCC（2.7~5.5V 均可）
        │
        └──[负极]──→ 所有模块公共 GND 汇流点
```

> **重要**：所有模块 GND 必须短接至同一 GND 汇流点，否则 UART 信号将出现偏移或误码。
>
> 若 2M16S 板载 5V 输出电流不足（< 500 mA），可在电池正极另引一路，经独立 LM2596 / MP1584 模块降压至 5V，为 UNO 及激光模组单独供电。

## 3. 游戏手柄解码器（YFRobot 2015）→ 2M16S 接线

### 3.1 YFRobot 2015 解码器引脚说明

YFRobot 2015 是蓝牙游戏手柄的硬件解码模块，将蓝牙信号转换为 **PS2 协议**（CLK/CS/CMD/DAT 四线 SPI-like 接口）。

| 解码器引脚 | 描述 | 连接至 2M16S |
|:---------:|------|:-----------:|
| **GND** | 电源负极 | GND |
| **VCC** | 电源正极（5V） | 5V |
| **CLK** | PS2 时钟（Clock，主控输出） | **Pin 10** |
| **CS** | PS2 片选（Chip Select / ATT，主控输出） | **Pin 11** |
| **CMD** | PS2 命令（MOSI，主控→手柄） | **Pin 12** |
| **DAT** | PS2 数据（MISO，手柄→主控） | **Pin 13** |

> **协议说明**：CLK 由 2M16S 主控输出，DAT/CMD 在每个时钟边沿采样。2M16S 固件通过 PS2 库（`PS2X_lib`）读取摇杆轴值（0~255，中位 128）及所有按键状态。

### 3.2 2M16S 板引脚对外连接汇总

| 2M16S 引脚 | 描述 | 连接至 |
|:----------:|------|:------:|
| **VIN / VM** | 电机电源输入（7~18V） | 电池正极（~11.1V） |
| **VCC** | 逻辑电源（5V） | 板载稳压自给 |
| **GND** | 公共地 | 系统公共 GND |
| **Pin 10** | PS2 CLK 输出 | 解码器 **CLK** |
| **Pin 11** | PS2 CS 输出 | 解码器 **CS** |
| **Pin 12** | PS2 CMD 输出 | 解码器 **CMD** |
| **Pin 13** | PS2 DAT 输入 | 解码器 **DAT** |
| **M1A / M1B** | 左电机输出端1 / 端2 | 左电机正极（红）/ 负极（黑） |
| **M2A / M2B** | 右电机输出端1 / 端2 | 右电机正极（红）/ 负极（黑） |
| **TX** | UART 串口发送 | Digital PWM UNO **Pin 4**（SoftSerial RX） |
| **RX** | UART 串口接收（可选） | Digital PWM UNO **Pin 5**（SoftSerial TX） |
| **S1~S16** | 16路舵机 PWM 输出（50 Hz） | 舵机（本项目暂未使用，可扩展） |

## 5. 2M16S → Digital PWM UNO 接线

### 5.1 电平兼容性

> 2M16S TX 输出为 **5V TTL**，Digital PWM UNO Pin 4 亦为 5V 逻辑，**可直接连接，无需电平转换**。

| 2M16S 引脚 | 方向 | Digital PWM UNO 引脚 | 说明 |
|:----------:|:----:|:-------------------:|------|
| **TX** | → | **Pin 4**（SoftwareSerial RX） | 手柄状态帧发送至 UNO |
| **RX**（可选） | ← | **Pin 5**（SoftwareSerial TX） | UNO 回传激光命中次数 |
| **5V** | → | **5V** | 为 UNO 及激光模组供电 |
| **GND** | — | **GND** | 共地 |

> 使用 SoftwareSerial（Pin 4 / Pin 5）而非硬件 UART（Pin 0 / Pin 1），避免烧录固件时串口冲突。

## 6. Digital PWM UNO → 激光系统接线

### 6.1 激光发射器接线

| 模块引脚 | 连接至 UNO 引脚 | 说明 |
|:--------:|:--------------:|------|
| **VCC** | **5V** | 模块工作电压 |
| **GND** | **GND** | 共地 |
| **TTL/Signal** | **Pin 9**（OC1A，Timer1） | 产生 38 kHz PWM 载波调制信号 |

> **频率说明**：激光发射器最高支持 100 kHz，这里设置为 **38 kHz** 以匹配 VS1838B 的解调频率。

| **Pin 1（OUT）** | **Pin 2**（INT0） | 解调输出（Active LOW），下降沿触发命中中断 |
| **Pin 2（GND）** | **GND** | 公共地 |
| **Pin 3（VCC）** | **5V** | 工作电压（2.7~5.5V 均可） |

## 7. 运动控制逻辑

### 7.1 游戏手柄按键 → 动作映射

| 手柄操作 | 动作 | 左摇杆 Y | 左摇杆 X | 右摇杆 X |
|:-------:|:----:|:--------:|:--------:|:--------:|
| 左摇杆 **上推** | 前进 | + | 0 | 0 |
| 左摇杆 **下推** | 后退 | − | 0 | 0 |
| 左摇杆 **左上** | 左前（左弧线前进） | + | − | 0 |
| 左摇杆 **右上** | 右前（右弧线前进） | + | + | 0 |
| 左摇杆 **左下** | 左后（左弧线后退） | − | − | 0 |
| 左摇杆 **右下** | 右后（右弧线后退） | − | + | 0 |
| 右摇杆 **左推** | 原地左转 | 0 | 0 | − |
| 右摇杆 **右推** | 原地右转 | 0 | 0 | + |
| 双摇杆 **归中** | 滑行停止（惰行） | 0 | 0 | 0 |
| 按下 **L1 键** | 制动停止（急停） | — | — | — |
| 按下 **L2 键** | 激光发射 | — | — | — |

### 7.2 差速控制算法（2M16S 板载固件）

PS2 摇杆原始值范围 0~255，中位 128。2M16S 固件按如下逻辑计算左右电机速度：

```
// 摇杆原始值（0~255，中位 128）
ry  = PS2_LY          // 左摇杆 Y（128→0 = 推杆前进方向）
rx  = PS2_LX          // 左摇杆 X（0→128 = 推杆左方向）
rx2 = PS2_RX          // 右摇杆 X（用于原地转）

// 归一化至 -100 ~ +100
speed = (128 - ry)  * 100 / 128   // 正=前进，负=后退
turn  = (rx  - 128) * 100 / 128   // 正=右，负=左
turn2 = (rx2 - 128) * 100 / 128   // 右摇杆，正=右原地转

// 差速计算
if abs(turn2) > DEADZONE:          // 右摇杆：原地转
    leftSpeed  = -turn2
    rightSpeed = +turn2
else:                              // 左摇杆：弧线运动
    leftSpeed  = speed - turn
    rightSpeed = speed + turn

// 映射到 PWM 0~255，并限幅
leftPWM  = clamp(abs(leftSpeed)  * 255 / 100, 0, 255)
rightPWM = clamp(abs(rightSpeed) * 255 / 100, 0, 255)
```

### 7.3 电机控制真值表

> 定义：M1 = 左电机，M2 = 右电机；speed = 摇杆换算后的 PWM（0~255）；k = 差速系数（0~1）

| 动作 | 左电机（M1） | M1A | M1B | M1 PWM | 右电机（M2） | M2A | M2B | M2 PWM |
|:----:|:-----------:|:---:|:---:|:------:|:-----------:|:---:|:---:|:------:|
| **前进** | 正转 | HIGH | LOW | speed | 正转 | HIGH | LOW | speed |
| **后退** | 反转 | LOW | HIGH | speed | 反转 | LOW | HIGH | speed |
| **左前（左弧线）** | 慢正 | HIGH | LOW | speed×k | 正转 | HIGH | LOW | speed |
| **右前（右弧线）** | 正转 | HIGH | LOW | speed | 慢正 | HIGH | LOW | speed×k |
| **左后（左弧线）** | 慢反 | LOW | HIGH | speed×k | 反转 | LOW | HIGH | speed |
| **右后（右弧线）** | 反转 | LOW | HIGH | speed | 慢反 | LOW | HIGH | speed×k |
| **原地左转** | 反转 | LOW | HIGH | 200 | 正转 | HIGH | LOW | 200 |
| **原地右转** | 正转 | HIGH | LOW | 200 | 反转 | LOW | HIGH | 200 |
| **制动停止（L1）** | 制动 | HIGH | HIGH | 0 | 制动 | HIGH | HIGH | 0 |
| **滑行停止** | 惰行 | LOW | LOW | 0 | 惰行 | LOW | LOW | 0 |

> **差速系数 k**：弧线转向时内侧电机速度 = `speed × (1 − |X 轴偏移量| / 100)`，实现平滑弧线。
> **制动停止**（L1 键）：M1A = M1B = HIGH → H 桥短路制动，电机迅速锁止；松开 L1 后恢复摇杆控制。
> **滑行停止**（摇杆归中）：M1A = M1B = LOW → H 桥开路，电机自然惰行至停。

## 8. UART 串口通信协议（2M16S → Digital PWM UNO）

### 8.1 物理层参数

| 参数 | 值 |
|:----:|:--:|
| 波特率 | 115200 bps |
| 数据位 | 8 bit |
| 停止位 | 1 bit |
| 校验 | 无 |
| UNO 接口 | SoftwareSerial（Pin 4 RX / Pin 5 TX） |
| 电平兼容 | 5V TTL 直连（双方均为 5V 逻辑） |

### 8.2 手柄状态帧（2M16S → UNO，按行发送）

```json
{"lspd":200,"rspd":200,"btn":0}        // 前进（全速），无按键
{"lspd":-200,"rspd":-200,"btn":0}      // 后退
{"lspd":80,"rspd":200,"btn":0}         // 左前差速弧线
{"lspd":200,"rspd":80,"btn":0}         // 右前差速弧线
{"lspd":80,"rspd":-200,"btn":0}        // 左后差速弧线
{"lspd":-200,"rspd":80,"btn":0}        // 右后差速弧线
{"lspd":-150,"rspd":150,"btn":0}       // 原地左转
{"lspd":150,"rspd":-150,"btn":0}       // 原地右转
{"lspd":0,"rspd":0,"btn":0}            // 滑行停止（摇杆归中）
{"lspd":0,"rspd":0,"btn":1}            // L1 按下（制动停止）
{"lspd":0,"rspd":0,"btn":4}            // L2 按下（激光发射）
```

> `lspd` / `rspd`：−255 ~ +255，正值 = 前进方向，负值 = 后退方向。
> `btn`：按键位字段，Bit0 = L1（制动），Bit2 = L2（激光触发），可按需扩展其他按键。
> 每帧以 `\n`（换行符）结尾，UNO 按行读取并解析 JSON。

### 8.3 状态回传帧（UNO → 2M16S，可选）

```json
{"ir_count":5}      // 激光命中次数
```

## 10. 具体接线步骤（Step-by-Step）

### 第一步：搭建供电电路

1. 将三节 18650 电池**串联**（第一节负极接第二节正极，第二节负极接第三节正极），总电压约 11.1V。
2. 在正极引线上串入 **ON/OFF 拨动开关**（方便整体断电，开关靠近电池侧）。
3. 正极（经开关）→ **2M16S 板 VIN / VM**（电机电源输入端子）。
4. 负极 → **2M16S 板 GND**（公共地汇流点）。
5. 确保开关处于 **OFF** 状态，暂不通电。

### 第二步：YFRobot 2015 解码器 → 2M16S（PS2 接线，6 根线）

6. 解码器 **GND** → 2M16S **GND**（公共地）
7. 解码器 **VCC** → 2M16S **5V**（板载 5V 输出）
8. 解码器 **CLK** → 2M16S **Pin 10**
9. 解码器 **CS** → 2M16S **Pin 11**
10. 解码器 **CMD** → 2M16S **Pin 12**
11. 解码器 **DAT** → 2M16S **Pin 13**

> CLK / CS / CMD 均由 2M16S 主控输出，DAT 由解码器（手柄端）输入至 2M16S，均为 5V TTL，直接连接。
> 若 DAT 线信号不稳定，可在 DAT 与 VCC 之间加 10 kΩ 上拉电阻。

### 第三步：2M16S → 电机（4 根线）

12. **M1A** → 左电机**正极线（红）**
13. **M1B** → 左电机**负极线（黑）**
14. **M2A** → 右电机**正极线（红）**
15. **M2B** → 右电机**负极线（黑）**

> 调试时若某电机转向相反，**互换该电机两根接线**即可，无需修改代码。

### 第四步：2M16S → Digital PWM UNO（串口 + 供电，4 根线）

16. 2M16S **TX** → UNO **Pin 4**（SoftwareSerial RX，接收手柄状态帧）
17. 2M16S **RX** ← UNO **Pin 5**（SoftwareSerial TX，可选，回传命中次数）
18. 2M16S **5V** → UNO **5V**（为 UNO 供电）
19. 2M16S **GND** → UNO **GND**（共地）

### 第五步：激光发射器接线（3 根线）

20. 激光模组 **VCC** → UNO **5V**
21. 激光模组 **GND** → 公共 GND
22. 激光模组 **TTL / Signal** → UNO **Pin 9**（Timer1 OC1A，产生 38 kHz PWM）

### 第六步：VS1838B 激光接收器接线（3 根线）

> VS1838B 正面（圆弧面朝向目标），引脚从左到右：OUT、GND、VCC。

23. VS1838B **Pin 3（VCC）** → UNO **5V**
24. VS1838B **Pin 2（GND）** → 公共 GND
25. VS1838B **Pin 1（OUT）** → UNO **Pin 2**（INT0 外部中断，Active LOW）

### 第七步：通电测试顺序

26. 再次逐一检查所有接线，确认无短路、无反接、无虚接。
27. 将电源开关拨至 **ON**，2M16S 板、解码器、Digital PWM UNO 同步上电。
28. 蓝牙手柄开机（长按 HOME / PS 键），等待与**解码器**自动配对——手柄指示灯常亮 = 配对成功。
29. 配对成功后，**缓慢推动左摇杆（上/下）**，观察左右电机是否同向转动。
   - 若某电机方向相反 → 断电，交换该电机两根接线 → 重新通电验证。
30. 推动左摇杆**斜向左上**，确认左侧电机速度小于右侧（左弧线前进）；斜向右上则相反。
31. 推动右摇杆**左/右**，确认两电机反向转动（原地左转 / 原地右转）。
32. 摇杆完全归中，电机应**惰行至停止**（滑行停止）。
33. 按下 **L1 键**，电机应**立即锁止**（制动停止）；松开后恢复摇杆控制。
34. 按下 **L2 键**，用手机摄像头对准激光发射器，画面中应出现亮点（980 nm 红外可被摄像头捕获）。
35. 将激光对准 VS1838B，UNO Serial Monitor（115200 bps）应显示命中次数递增。

## 11. 注意事项与安全提醒

| 序号 | 事项 | 说明 |
|:----:|------|------|
| 1 | **必须共地** | 电池、2M16S、UNO、激光模组的所有 GND 必须接在同一汇流点，否则 UART 逻辑电平偏移导致误码 |
| 2 | **UART 5V 直连** | 2M16S TX 与 UNO Pin 4 均为 5V TTL 逻辑，可直连，无需电平转换 |
| 3 | **烧录时断串口** | 烧录 UNO 固件前，断开 Pin 4（SoftSerial RX）以防外部信号干扰上传 |
| 4 | **电机转向校正** | 若某路电机转向相反，交换该路电机输出线（M1A↔M1B 或 M2A↔M2B），不要修改固件 |
| 5 | **H 桥电流额定** | 2M16S 内置 H 桥一般额定 2A / 路，启动瞬间电流可达 3~4A；建议实测堵转电流确认不超额定值 |
| 6 | **电机供电电压** | 电机额定 12V，3 节 18650 满电 ~12.6V，标称 ~11.1V 可满足额定转速；2 节（~7.4V）约降速 40% |
| 7 | **激光安全** | 980 nm 30 mW 近红外激光**不可见但有害**，严禁直视或对准他人眼睛 |
| 8 | **激光频率匹配** | 发射器 TTL 必须设置为 **38 kHz**，否则 VS1838B 无法正确解调并输出命中信号 |
| 9 | **手柄配对** | 2M16S 上电后进入配对模式，手柄按 HOME 键开始配对；已配对手柄重新上电后可自动重连 |
| 10 | **UART 超时保护** | UNO 固件建议实现接收超时（推荐 500 ms），超时未收到帧则自动发出滑行停止指令 |
