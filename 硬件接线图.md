## 1. 供电电路

```
18650 电池组（2节串联 标称~7.4V，满电~8.4V）
        │
        ├──[正极]──┬──────────────────────→ TB6612 VM（电机驱动电源）
        │          └──→ 5V 降压模块（LM2596/MP1584）──→ ESP32-S3 5V 引脚
        │
        └──[负极]──────────────────────── 共地 GND（TB6612 GND + ESP32 GND）
```

> **重要说明**
> - 2节 18650 串联标称 7.4V，电机额定 12V，实际转速约为额定的 62%，扭矩按比例降低，仍可正常工作
> - ESP32-S3 **必须**通过独立 5V 降压模块供电，切勿与电机共用供电线（电机启动浪涌电流会导致 ESP32 复位）
> - TB6612 D153C 模块的 VCC 接 ESP32 的 **3.3V** 引脚（逻辑电源）

## 2. ESP32-S3 N8R2 → TB6612 D153C 接线

### TB6612 D153C 模块引脚定义（正面示意）

```
┌──────────────────────────────────────────┐
│        TB6612 D153C 模块正面              │
│                                          │
│  [VM] [GND] [VCC] [STBY]  ← 左侧引脚    │
│                                          │
│  [AO1] [AO2]    [BO1] [BO2]  ← 电机输出 │
│                                          │
│  [PWMA][AIN1][AIN2]  [PWMB][BIN1][BIN2] │
│         ↑  控制引脚  ↑                   │
└──────────────────────────────────────────┘
```

### 接线对照表

| TB6612 D153C 引脚 | 连接至 | ESP32-S3 引脚/电源 | 说明 |
|:-----------------:|:------:|:-----------------:|------|
| **VM** | → | 电池正极（~7.4V） | 电机驱动电源（最大15V） |
| **VCC** | → | 3.3V | 逻辑电源 |
| **GND** | → | GND | 共地 |
| **STBY** | → | GPIO 10 | 驱动使能（HIGH=工作，LOW=休眠）|
| **PWMA** | → | GPIO 4 | 左电机（A）PWM 速度控制 |
| **AIN1** | → | GPIO 5 | 左电机（A）方向控制引脚1 |
| **AIN2** | → | GPIO 6 | 左电机（A）方向控制引脚2 |
| **PWMB** | → | GPIO 7 | 右电机（B）PWM 速度控制 |
| **BIN1** | → | GPIO 8 | 右电机（B）方向控制引脚1 |
| **BIN2** | → | GPIO 9 | 右电机（B）方向控制引脚2 |

## 3. TB6612 D153C → 电机（33GB-520-18.7F）接线

| TB6612 引脚 | 连接至 | 说明 |
|:-----------:|:------:|------|
| **AO1** | → 左电机 正极线（红） | 电机 A 输出端1 |
| **AO2** | → 左电机 负极线（黑） | 电机 A 输出端2 |
| **BO1** | → 右电机 正极线（红） | 电机 B 输出端1 |
| **BO2** | → 右电机 负极线（黑） | 电机 B 输出端2 |

> 若调试时电机实际转向与预期相反，**互换该电机的两根输出线**即可（例如 AO1/AO2 对调），无需修改代码。

## 4. 激光发射器接线（18×45 980nm 30mW TTL调制模组）

### 模块引脚（通常为3线）

| 激光发射器引脚 | 颜色 | 连接至 | ESP32-S3 引脚/电源 | 说明 |
|:-------------:|:----:|:------:|:-----------------:|------|
| **VCC** | 红 | → | 5V（降压模块输出）| 模组工作电压 5V |
| **GND** | 黑 | → | GND | 共地 |
| **TTL / Signal** | 黄/白 | → | GPIO 11 | PWM 调制信号输入 |

> **信号说明**
> - ESP32 GPIO 输出 3.3V，TTL 标准 VIH ≥ 2.0V，**无需电平转换**，可直连
> - GPIO 11 输出 **38 KHz PWM 方波**（占空比 50%）驱动激光以匹配 VS1838B 接收器
> - `HIGH`（PWM 输出）= 激光开启；`LOW`（停止 PWM）= 激光关闭

## 5. 激光接收器接线（VS1838B / 1838）

### VS1838B 引脚图（正面引脚朝下，凸面朝向自己）

```
     ┌──────────┐
     │  VS1838B │
     │  (1838)  │
     └──┬──┬──┬─┘
        │  │  │
       (1) (2) (3)
       OUT GND VCC
```

| 1838 引脚 | 编号 | 连接至 | ESP32-S3 引脚/电源 | 说明 |
|:---------:|:----:|:------:|:-----------------:|------|
| **OUT** | 引脚1（左）| → | GPIO 12 | 信号输出（Active LOW） |
| **GND** | 引脚2（中）| → | GND | 共地 |
| **VCC** | 引脚3（右）| → | 3.3V | 工作电压（2.7~5.5V）|

> **工作原理**
> - VS1838B 内置 **38 KHz 带通滤波器**，激光发射器以 38 KHz 调制时可被正常接收
> - 接收到有效信号时 OUT 输出**低电平**（Active LOW），ESP32 读取 LOW 即表示检测到激光
> - 用于**激光对射**场景：激光束被遮断时，OUT 从 LOW 变为 HIGH，触发 ESP32 中断或轮询检测

## 6. ESP32-S3 N8R2 完整 GPIO 分配表

| GPIO 编号 | 方向 | 功能类型 | 连接设备 | 备注 |
|:---------:|:----:|:--------:|----------|------|
| **GPIO 4** | 输出 | LEDC PWM | TB6612 PWMA | 左电机速度（0~255） |
| **GPIO 5** | 输出 | Digital OUT | TB6612 AIN1 | 左电机方向1 |
| **GPIO 6** | 输出 | Digital OUT | TB6612 AIN2 | 左电机方向2 |
| **GPIO 7** | 输出 | LEDC PWM | TB6612 PWMB | 右电机速度（0~255） |
| **GPIO 8** | 输出 | Digital OUT | TB6612 BIN1 | 右电机方向1 |
| **GPIO 9** | 输出 | Digital OUT | TB6612 BIN2 | 右电机方向2 |
| **GPIO 10** | 输出 | Digital OUT | TB6612 STBY | 驱动器使能 |
| **GPIO 11** | 输出 | LEDC PWM | 激光发射器 TTL | 38 KHz 载波 |
| **GPIO 12** | 输入 | Digital IN + 中断 | VS1838B OUT | 激光接收检测 |
| **GPIO 43** | 输出 | UART0 TX | USB 调试串口 | 保留 |
| **GPIO 44** | 输入 | UART0 RX | USB 调试串口 | 保留 |
| **内置 WiFi** | 双向 | 802.11 b/g/n | 安卓手机 | AP 或 STA 模式 |

> **N8R2 禁用 GPIO 说明**
> - GPIO 26~32：Quad Flash 占用，**禁止使用**
> - GPIO 19、20：USB OTG D-/D+，避免使用
> - GPIO 0：BOOT 按键，避免上拉/下拉冲突

## 8. 电机运动控制逻辑真值表

> **定义**：电机 A = 左电机，电机 B = 右电机
> **PWM**：255 = 全速，0 = 停止，可设中间值做差速转弯

| 动作 | 左电机（A） | AIN1 | AIN2 | PWMA | 右电机（B） | BIN1 | BIN2 | PWMB |
|:----:|:-----------:|:----:|:----:|:----:|:-----------:|:----:|:----:|:----:|
| **前进** | 正转 | HIGH | LOW | 255 | 正转 | HIGH | LOW | 255 |
| **后退** | 反转 | LOW | HIGH | 255 | 反转 | LOW | HIGH | 255 |
| **左前**（左弧线前进）| 停止 | LOW | LOW | 0 | 正转 | HIGH | LOW | 255 |
| **右前**（右弧线前进）| 正转 | HIGH | LOW | 255 | 停止 | LOW | LOW | 0 |
| **左后**（左弧线后退）| 停止 | LOW | LOW | 0 | 反转 | LOW | HIGH | 255 |
| **右后**（右弧线后退）| 反转 | LOW | HIGH | 255 | 停止 | LOW | LOW | 0 |
| **原地左转** | 反转 | LOW | HIGH | 200 | 正转 | HIGH | LOW | 200 |
| **原地右转** | 正转 | HIGH | LOW | 200 | 反转 | LOW | HIGH | 200 |
| **制动停止** | 制动 | HIGH | HIGH | 0 | 制动 | HIGH | HIGH | 0 |
| **滑行停止** | 惰行 | LOW | LOW | 0 | 惰行 | LOW | LOW | 0 |

> **STBY（GPIO 10）**：必须 `HIGH` 电机才能工作；置 `LOW` 可紧急切断双路电机

## 9. 安卓手机通信方案（WiFi TCP/WebSocket）

### 推荐方案：ESP32 作为 WiFi AP（比赛环境首选）

```
安卓手机 APP
    │
    └──── WiFi ────→ ESP32-S3（AP 模式）
                      │  SSID: Robot-ESP32
                      │  IP:   192.168.4.1
                      │  Port: 8080（TCP / WebSocket）
                      │
                      ├─── 解析指令 ──→ 控制电机（GPIO 4~10）
                      ├─── 解析指令 ──→ 控制激光（GPIO 11）
                      └─── 读取状态 ←── 激光接收（GPIO 12）
```

### 控制指令格式（JSON over TCP）

```json
{ "cmd": "forward",       "speed": 200 }
{ "cmd": "backward",      "speed": 200 }
{ "cmd": "left_forward",  "speed": 180 }
{ "cmd": "right_forward", "speed": 180 }
{ "cmd": "left_backward", "speed": 180 }
{ "cmd": "right_backward","speed": 180 }
{ "cmd": "spin_left",     "speed": 150 }
{ "cmd": "spin_right",    "speed": 150 }
{ "cmd": "stop" }
{ "cmd": "laser_on" }
{ "cmd": "laser_off" }
```

### 状态上报格式（ESP32 → 安卓）

```json
{ "laser_hit": true,  "timestamp": 12345 }
{ "laser_hit": false, "timestamp": 12389 }
```

## 10. 注意事项与安全提醒

| 序号 | 事项 | 说明 |
|:----:|------|------|
| 1 | **必须共地** | 电池组、TB6612、ESP32、激光模组的 GND 必须全部短接 |
| 2 | **独立供电 ESP32** | 使用 LM2596 或 MP1584 降压模块从电池取 5V 给 ESP32，防止电机启动浪涌导致复位 |
| 3 | **滤波电容** | VM 与 GND 之间并联 100µF 电解电容 + 0.1µF 陶瓷电容，抑制电机反电动势干扰 |
| 4 | **激光安全** | 980nm 30mW 近红外激光对眼睛有害，**严禁直视**激光束 |
| 5 | **电池放电倍率** | 两个 12V 电机在 7.4V 下启动电流可达 2~4A，确认 18650 电池持续放电倍率 ≥ 2C |
| 6 | **N8R2 GPIO 限制** | GPIO 26~32 被 Flash/PSRAM 占用，**禁止**用作普通 IO |
| 7 | **STBY 控制** | 若无需动态控制休眠，可将 STBY 直接接 3.3V 常高，GPIO 10 可释放 |
| 8 | **激光调制频率** | 激光发射器调制频率须设为 **38 KHz** 方可被 VS1838B 正确解调 |
